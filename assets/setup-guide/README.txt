Setup Guide Images
==================

Place screenshot images here to show them in the native GUI setup wizard.
Images are optional - the wizard works without them but looks better with visual guides.

Supported filenames:
  step1-token.png    - Shows on the "Discord Bot Token" step
                       Recommended: Screenshot of Discord Developer Portal > Bot page
                       showing where the token/Reset Token button is
  step3-ai.png       - Shows on the "AI Features" step
                       Recommended: Screenshot of OpenRouter or Gemini API key pages

Image specs:
  - Format: PNG (recommended), JPG also works
  - Size: Will be scaled to fit ~545x310 pixels (step1) or ~545x170 (step3)
  - Keep images clear and readable at the display size

For exe builds, these images are automatically bundled inside the executable
and extracted at runtime.
