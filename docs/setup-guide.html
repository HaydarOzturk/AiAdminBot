🛡️ AiAdminBot

AI-Powered Discord Server Administration — Setup Guide

1 Prerequisites

Before you start, make sure you have:

  • Node.js 18+ — Download here (not needed for .exe)
  • A Discord account with a server you own (or admin access)
  • (Optional) An OpenRouter or Google Gemini API key for AI features

2 Creating a Discord Bot

  1. Go to the Discord Developer Portal
  2. Click "New Application" in the top right
  3. Give your bot a name (e.g. "AiAdminBot") and click Create
  4. Go to the Bot tab in the left sidebar
  5. Click "Reset Token" and copy the token
⚠️ Important: Never share your bot token! Anyone with the token can control your bot.

Enable Privileged Intents

Still on the Bot tab, scroll down to Privileged Gateway Intents and enable:

  • ✅ Presence Intent
  • ✅ Server Members Intent
  • ✅ Message Content Intent

Click Save Changes.

3 Getting AI API Keys (Optional)

AI features (chat assistant and smart moderation) are free but require an API key.

OpenRouter (recommended for free models)
  1. Go to openrouter.ai/keys
  2. Sign up or log in
  3. Click "Create Key"
  4. Copy the key (starts with sk-or-)

OpenRouter provides free models — no credit card needed.

Google Gemini (alternative free option)
  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the key

Gemini has a generous free tier. You can configure both providers for automatic failover.

4 Installation

Option A: Standalone Executable (.exe)

  1. Download the latest release from GitHub Releases
  2. Run AiAdminBot.exe
  3. The setup wizard will guide you through configuration
  4. Run the .exe again to start the bot

Option B: From Source

git clone https://github.com/HaydarOzturk/AiAdminBot.git
cd discord-admin-bot
npm install
npm run setup     # Interactive setup wizard
npm run deploy    # Register slash commands
npm start         # Start the bot

5 Inviting the Bot

After setup, the wizard shows your invite link. You can also build it manually:

https://discord.com/oauth2/authorize?client_id=YOUR_CLIENT_ID&scope=bot+applications.commands&permissions=8

Replace YOUR_CLIENT_ID with your bot's Application ID (found in the Developer Portal → General Information).

Tip: The permission 8 means Administrator. AiAdminBot needs admin to manage channels, roles, and moderation.

8 Troubleshooting

Bot doesn't come online
  • Check that your DISCORD_TOKEN in .env is correct
  • Make sure all 3 Privileged Intents are enabled in the Developer Portal
  • Check the logs/ folder for error details
Slash commands don't appear
  • Run npm run deploy (or /sync commands in Discord)
  • Wait a few minutes — Discord caches commands globally
  • Try restarting Discord (Ctrl+R)
AI features don't work
  • Verify your API key is correct in .env
  • Make sure AI_CHAT_ENABLED=true and/or AI_MODERATION_ENABLED=true
  • OpenRouter free models may have rate limits — try again later
  • Check the logs for error messages
Messages older than 14 days can't be deleted

This is a Discord API limitation. The /clear command can only bulk-delete messages newer than 14 days. Older messages must be deleted individually.

Bot can't timeout/kick/ban a user
  • The bot's role must be higher than the target user's highest role
  • Run /fix-permissions to auto-fix the bot's role position
  • Server owners cannot be timed out by anyone
